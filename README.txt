Anleitung zum Start des Simulators:

NodeJS und eine JRE müssen installiert sein. (https://nodejs.org/en/download/ https://www.java.com/de/download/) 

Auf der Kommandozeile:
1. In dieses Verzeichnis ("autorensystem") wechseln.
2. Den Server für das Autorensystem mit dem Befehl "node server.js" starten.
3. Im Browser die Adresse "localhost:3000" aufrufen.
4. Für die Simulation in das Unterverzeichnis "rulegenerator" wechseln.
5. Den Regelgenerator mit dem Befehl "java -jar rulegenerator.jar" starten.
6. Im Autorensystem ein Szenario anlegen und speichern und ggf. die Seite einmal neu laden.
7. Fertig!
